Labels Documentation

Labels are generated from Metrics and written to GMC_Feed. Output columns are controlled from Settings > Label Settings.

Main Labels

LabelMeaningDefault Output
Revenue Simplehas_revenue or no_revenue based on store revenue in the selected timeframe.custom_label_0
Price IntervalGroups products into price buckets using the configured price step.custom_label_1
GAds ROASLabels Ads ROAS using Good ROAS and Bad ROAS thresholds.custom_label_2
GAds CVRLabels conversion rate using Good CVR and Bad CVR thresholds.custom_label_3
GAds ClicksLabels click volume using High Clicks and Low Clicks thresholds.custom_label_4

Optional Labels

LabelMeaning
Revenue AdvancedCompares each product revenue against account average revenue.
OrdersClassifies products by order count using the High Orders threshold.
Available VariantsGroups related variants when the source platform provides variant-level rows.
TrendCompares recent 14-day revenue against the older part of the selected timeframe.
New ProductMarks products newer than the configured New Product Days window.
Performance IndexCombines store revenue, Ads cost, ROAS, orders, and price into a performance classification.

Important Notes

  • Leave an output mapping blank to disable that label column.
  • Google Ads labels require matching IDs between GAds.id and the platform Product ID.
  • Duplicate platform IDs are merged before labels are calculated, so each feed ID should have one final row.
  • GMC_Feed_2 uses the same labels as GMC_Feed, but can use alternate IDs when configured.
