Labels Documentation

Labels are generated from Metrics and written to GMC_Feed. Output columns are controlled from Settings > Label Settings.

Main Labels

LabelMeaningPossible ValuesDefault Output
Revenue Simple Shows whether the product has store revenue in the selected timeframe. has_revenueno_revenue custom_label_0
Price Interval Groups products into price buckets using the configured price step. price_0_50price_50_100price_100_150 etc. based on the configured interval. Invalid prices output invalid_price_data. custom_label_1
GAds ROAS Labels Ads ROAS using Good ROAS and Bad ROAS thresholds. high_roasavg_roaslow_roasno_roas custom_label_2
GAds CVR Labels conversion rate using Good CVR and Bad CVR thresholds. high_cvravg_cvrlow_cvrno_cvr custom_label_3
GAds Clicks Labels click volume using High Clicks and Low Clicks thresholds. high_clicksavg_clickslow_clicksno_clicks custom_label_4

Optional Labels

LabelMeaningPossible Values
Revenue Advanced Compares each product revenue against account average revenue. high_revenueavg_revenuelow_revenueno_revenue
Orders Classifies products by order count using the High Orders threshold. high_ordersaverage_ordersone_orderno_orders
Available Variants Shows in-stock variants over total variants for related products. Dynamic ratio such as 0/31/32/33/3. Blank when no ID/parent can be resolved.
Trend Compares recent 14-day revenue against the older part of the selected timeframe. up_trendstable_trenddown_trendrecent_activity_onlyno_trend
New Product Marks products newer than the configured New Product Days window. new_productinvalid_date or blank for older products.
Performance Index Combines store revenue, Ads cost, ROAS, orders, and price into a performance classification. INDEXNEAR-INDEXLOW-INDEXNO-INDEXEXCLUDE-INDEX
Site ROAS Numeric helper output used by Performance Index. A numeric value such as 0.002.3512.80.

Important Notes

  • Leave an output mapping blank to disable that label column.
  • Google Ads labels require matching IDs between GAds.id and the final feed ID.
  • Product ID Matching can keep the Default Feed ID or use a selected platform fallback ID when it matches GAds.id.
  • Duplicate platform IDs are merged before labels are calculated, so each feed ID should have one final row.
  • GMC_Feed_2 uses the same labels as GMC_Feed, but can use alternate IDs when configured.
