üõ†Ô∏è 2. Google Ads Setup

To get ROAS and Clicks data:

  1. Copy the code from google_ads_script_export.js.
  2. Go to Google Ads > Tools > Scripts.
  3. Paste the code and set the SPREADSHEET_URL variable.
  4. Schedule it Daily (e.g., 4:00 AM).

‚ö° 3. Automation Setup

To enable the daily schedule:

  1. Open Apps Script Editor (Extensions > Apps Script).
  2. Open TriggerSetup.gs.
  3. Run setupShopifyComplete (or WooCommerce equivalent).

This schedules data fetching for 5:00 AM daily.
