⚡ Performance Labels

Automated Product Segmentation for Google Merchant Center & Meta

1. First-Time Setup

  • Open Performance Labels → Setup Guide
  • Choose your active platform: WooCommerce, Shopify, Gomag, or GA4
  • Complete the checklist: platform credentials, label settings, Google Ads script, and auto-fetch schedule

2. Configuration

  • Use Setup Guide → Tune Labels for ROAS, revenue, clicks, price intervals, timeframe, and ID prefix/suffix settings
  • Advanced users can still edit label defaults and FEED_EXPORT_MAPPING in Config.gs
  • Set a mapping to "" to disable that label calculation entirely

3. Running Manually

  • Performance Labels → Run Now — runs the active platform. Store platforms sync data and recalculate labels; GA4 runs the GA4 report only.
  • Performance Labels → Status & Logs — opens the Overview sheet
  • Performance Labels → Advanced → Platform Data Only — syncs the selected store platform without recalculating labels

4. Google Ads Setup

  • Open Performance Labels → Google Ads Script [To Copy] or GoogleAdsData.html from the GitHub repo
  • Go to Google Ads → Tools → Scripts and paste the code
  • Set the SPREADSHEET_URL variable to your spreadsheet URL
  • Schedule it daily at 4:00 AM (before the 5:00 AM store fetch)
⚠️ Do not paste the Google Ads script into this Sheet Script Editor — it belongs in Google Ads Scripts only.

5. Publishing the Feed (Google Merchant Center)

  • In your Spreadsheet: File → Share → Publish to web
  • Select the GMC_Feed tab → format: Comma-separated values (.csv) → Publish
  • Paste the generated URL into Google Merchant Center as your scheduled fetch URL

6. Facebook / Meta Feed

  • A second tab GMC_Feed_2 is auto-generated when ID_PREFIX or ID_SUFFIX is set in Config.gs
  • Example: ID_PREFIX = "wc_post_id_" produces IDs like wc_post_id_12345
  • Publish GMC_Feed_2 as CSV and use that URL in Meta Commerce Manager
💡 API Keys are stored securely in Google Script Properties — never in the code or spreadsheet.
