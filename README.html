⚡ Performance Labels

Automated Product Segmentation for Google Merchant Center & Meta

1. First-Time Setup

  • Set your platform in Config.gs: PLATFORM = 'woocommerce' or 'shopify'
  • Go to ⚙️ Settings → 🔑 Update Settings to enter your domain & API credentials
  • Click 🕐 Set Up Daily Auto-Fetch to enable automatic daily sync

2. Configuration

  • All thresholds are in Config.gs (ROAS, revenue, clicks, price intervals…)
  • Map labels to custom_label_0–custom_label_4 in FEED_EXPORT_MAPPING
  • Set a mapping to "" to disable that label calculation entirely

3. Running Manually

  • ▶️ Run All — fetches store data & recalculates all labels in one click
  • Fetch [Platform] Data — syncs data only, no label recalculation

4. Google Ads Setup

  • Open GoogleAdsData.gs from the GitHub repo
  • Go to Google Ads → Tools → Scripts and paste the code
  • Set the SPREADSHEET_URL variable to your spreadsheet URL
  • Schedule it daily at 4:00 AM (before the 5:00 AM store fetch)
⚠️ Do not paste GoogleAdsData.gs into this Script Editor — it belongs in Google Ads Scripts only.

5. Publishing the Feed (Google Merchant Center)

  • In your Spreadsheet: File → Share → Publish to web
  • Select the GMC_Feed tab → format: Comma-separated values (.csv) → Publish
  • Paste the generated URL into Google Merchant Center as your scheduled fetch URL

6. Facebook / Meta Feed

  • A second tab GMC_Feed_2 is auto-generated when ID_PREFIX or ID_SUFFIX is set in Config.gs
  • Example: ID_PREFIX = "wc_post_id_" produces IDs like wc_post_id_12345
  • Publish GMC_Feed_2 as CSV and use that URL in Meta Commerce Manager
💡 API Keys are stored securely in Google Script Properties — never in the code or spreadsheet.
